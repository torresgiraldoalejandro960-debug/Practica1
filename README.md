# Proyecto de Práctica - Desarrollo Web

**Autores:** Alejandro Torres

## Descripción

Proyecto de práctica para aprender y aplicar los fundamentos del desarrollo web frontend.

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Temas de HTML

- Estructura básica de un documento HTML
- Etiquetas semánticas (header, nav, section, article, footer)
- Formularios y validación
- Tablas
- Listas (ordenadas y desordenadas)
- Enlaces e imágenes
- Atributos y clases

## Temas de CSS

- Selectores (clase, id, etiqueta, atributo)
- Box Model (margin, padding, border)
- Flexbox
- Grid Layout
- Posicionamiento (static, relative, absolute, fixed)
- Transiciones y animaciones
- Media queries y diseño responsivo
- Pseudoclases y pseudoelementos

## Temas de JavaScript

- Variables y tipos de datos
- Operadores y expresiones
- Estructuras de control (if, else, switch)
- Bucles (for, while, do-while)
- Funciones y arrow functions
- Arrays y métodos de arrays
- Objetos y JSON
- DOM (Document Object Model)
- Eventos y event listeners
- Funciones asíncronas (Promises, async/await)
- Manipulación del DOM

## Objetivo

Aplicar los conocimientos adquiridos en proyectos prácticos y desarrollar habilidades de programación frontend.
