# Proyecto de Práctica Git

**Autor:** Alejandro Torres

## Descripción

Este es un proyecto de práctica para aprender los comandos básicos y avanzados de Git.

## Objetivo

Familiarizarse con:
- Comandos fundamentales de Git
- Manejo de ramas
- Resolución de conflictos
- Buenas prácticas de versionamiento

## Tecnologías

- Git
- GitHub
